# Ram Vinay Kumar - Portfolio

### Live - https://updated-portfolio-two-steel.vercel.app/

### Features -

- Modern Stack (React.js + Material UI)
- Minimal Design
- Easy To Browse
- Dark Mode
- Responsive Design

### Sections

- About
- Skills
- Experience
- Projects
- Education
- Contact

### Tech Stack Used -
- React.js
- Material UI
- Styled Components
- Email Js

### How To Use

- Clone this repo
- run `npm i`
- `npm start`

### How To Deploy -

- Login into Vercel with GitHub
- Select this repo
- Click Deploy
- Vercel will automatically deploy your website
